API Documentation

Paste It proviedes an API allowing to paste content via HTTP. More soon...

API Version: 01

Bash Script

The reay fun starts when using this little script: 
#!/bin/bash

URL="https://p.qrt.de"
API_VERSION="01"
LANG="txt"

if [ $# == 0 ]
then
	FILE="-"
else
	if [ -f "$1" ]
	then
		FILE="$1"
	else
		echo "Can't paste $1"
		exit -1
	fi
fi

API_URL="${URL}/api/v${API_VERSION}/add/"
ID=$(curl -s $API_URL -F "content=$(cat $FILE)" -F "language=${LANG}")
echo -n "${URL}/${ID}" | xclip && echo `xclip -o`
Note: There is a high chance that you might have to install xclip first.

Example Useage

Assuming you saved the script as ~/bin/pasteit and set chmod 700: 
$ echo "foobar" | pasteit
