About

Paste IT is a private paste bin service. It is an open source (MIT license) project and can be found at GitHub. Feel free to contribute, report bugs or give feedback.

Idea

There are a lot of paste bins on the internet but none suited my idea of having a plain, clean, fast and most importantly secure paste bin. And as an Linux user I need and want to paste and grep via command line. No CTRL+C CTRL+V shit. So an API was needed as well.

Features

  • SSL
  • Privacy: If you don't share the returning paste URL, nobody will read your paste. Not even Google. Well, as least as long as no one leaks our database :/
  • Infinite lifetime: If you don't add a expiry date, your paste will live long. Very long. But no guarantee. Sorry.
  • No external URLs, Ads or fancy social stuff.
  • No logging.

Gernal

Paste It uses Python, Django and Pygments.
